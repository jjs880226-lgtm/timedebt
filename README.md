# TimeDept ⏱️

시간 빚(Time Debt) 개념을 활용한 집중 관리 Android 애플리케이션입니다.

사용자가 해야 할 일을 등록하고 목표 시간을 설정하면,
집중 시간을 관리하고 자신의 시간 사용 습관을 확인할 수 있도록 도와주는 자기관리 앱입니다.

---

## 📌 프로젝트 소개

현대인들은 해야 할 일은 많지만 시간을 효율적으로 관리하지 못해
계획했던 일을 미루는 경우가 많습니다.

TimeDept는 이러한 문제를 해결하기 위해
'시간 빚'이라는 개념을 적용하여 사용자가 자신의 할 일을 관리하고
집중할 수 있도록 설계한 애플리케이션입니다.

---

## ✨ 주요 기능

### 1. 할 일(Task) 관리
- 해야 할 일 등록
- 할 일 목록 확인
- 완료 상태 관리
- 남은 시간 확인

### 2. 집중 타이머 기능
- 목표 시간 설정
- 집중 시간 측정
- 남은 시간 표시
- 진행률 표시

### 3. 시간 빚(Time Debt) 시스템
- 미완료 작업에 대한 시간 빚 생성
- 사용자의 시간 관리 습관 확인

### 4. 집중 관리 기능
- 집중 모드 제공
- 앱 사용 중 집중 시간 기록
- 효율적인 시간 관리 지원

### 5. 데이터 저장 기능
- SharedPreferences를 활용한 사용자 데이터 저장
- 앱 종료 후에도 데이터 유지
